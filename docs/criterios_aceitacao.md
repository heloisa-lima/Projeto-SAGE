# Critérios de aceitação
# História 1:
Critério: O sistema deve exibir uma lista com todos os locais cadastrados
Critério: Os espaços já reservados devem aparecer como indisponíveis

# História 2:
Critério: O sistema deve identificar quando uma nova reserva for feita.
Critério: O sistema deve enviar uma notificação para o cadastro do
funcionário no aplicativo

# História 3:
Critério: O sistema deve exibir uma lista das reservas realizadas

# História 4:
Critério: O sistema deve identificar, por meio do cadastro do usuário, se a
pessoa é um professor
Critério: Caso o usuário que solicitou a reserva seja um professor, o sistema
deve priorizar

# História 5:
Critério: O sistema deve permitir ao funcionário aceitar ou recusar
Critério: O funcionário deve ter acesso ao nome e às informações da reserva
para poder aceitar ou recusar.

# História 6:
Critério: O professor consegue alterar ou excluir apenas as reservas feitas
por ele.
Critério: Ao editar ou cancelar, a disponibilidade do espaço é atualizada.

# História 7:

# História 13:
Critério: O acesso só é permitido com credenciais institucionais válidas.
Critério: Caso a senha ou e-mail estejam incorretos, o sistema não permite o
login.

# História 14:
Critério: O sistema deve registrar a data, horário, nome do local, número da
chave e nome do responsável.
Critério: O sistema deve permitir marcar a devolução da chave com horário e
confirmação.

# História 15:
Critério: O sistema deve exibir a agenda do monitor vinculada à sala
específica.
Critério: Caso outro usuário tente reservar nesse mesmo horário, o sistema
deve indicar indisponibilidade.

# História 16:
Critério: O sistema deve armazenar todas as reservas realizadas com data,
horário e responsável.
Critério: O funcionário deve poder filtrar o histórico por sala, usuário ou
período.

# História 17:
Critério: O sistema deve gerar relatórios com número de reservas por espaço
e por tipo de usuário (professor, monitor, grêmio).
Critério: O relatório deve estar disponível em formato digital (PDF).

# História 18:

Critério: A notificação deve indicar claramente se a reserva foi aprovada ou
reprovada.
